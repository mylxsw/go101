Go 101

Go 101 is a book focusing on Go syntax/semantics and all kinds of details (Go 1.11 ready). This book also tries to help gophers gain a deep and thorough understanding of Go.

  • The book is maintained by @TapirLiu.
  • The go101.org website is hosted on App Engine. Surely, it is built with Go.
  • Code is hosted on both Gitlab and Github. You can also start a local go101 server for faster browsing by running either of the following two commands:
    • go get -u gitlab.com/go101/go101 && go101
    • go get -u github.com/go101/go101 && go101
NOTE: if your last go get -u command run was at before July 28th, 2018, please run this command again to install the latest go101 program and articles.
